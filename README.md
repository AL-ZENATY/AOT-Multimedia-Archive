# Attack on Titan Multimedia Archive

A multimedia-focused fan project dedicated to **Attack on Titan**.

This website was designed as both:

* an introduction for new viewers
* and a media archive for existing fans

The archive contains:

* Trailers
* Openings & Endings
* Soundtracks
* Character visuals
* Animated video backgrounds

## Features

* Dynamic multimedia interface
* Video-based visual design
* Responsive frontend structure
* Organized anime media archive
* Interactive navigation experience

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* SQL

## Warning ⚠️

This project contains major spoilers for Attack on Titan.

## Purpose

This project was created as a personal passion project and portfolio piece focused on combining:

* frontend design
* multimedia presentation
* backend functionality
* anime-inspired UI experiences
